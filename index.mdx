## What is Open LLM VTuber?

Open LLM VTuber is a **voice-interactive AI companion** with a Live2D animated avatar that runs completely offline on your computer. Inspired by AI VTubers like Neuro-sama, it brings together local LLMs, speech recognition, and text-to-speech in a fully modular, cross-platform application.

You can run it as a browser-based web app or as a desktop application with a transparent background **pet mode** — letting your AI companion float on top of other windows anywhere on your screen.

## Core capabilities

<AccordionGroup>
  <Accordion title="Real-time voice conversation">
    Speak naturally with your AI companion using hands-free voice interaction. The system automatically detects when you start and stop speaking using Voice Activity Detection (VAD), so there's no push-to-talk button required.
  </Accordion>
  <Accordion title="Voice interruption without headphones">
    Open LLM VTuber can be interrupted mid-sentence by your voice, even without headphones. The AI won't hear its own audio output, preventing feedback loops while maintaining natural conversation flow.
  </Accordion>
  <Accordion title="Live2D animated avatar">
    Your AI companion is represented by a fully animated Live2D avatar. The backend can control facial expressions and motions dynamically based on the conversation, using an emotion-to-expression mapping system.
  </Accordion>
  <Accordion title="Visual perception">
    The AI can see your screen or webcam feed. Enable screenshot capture, screen recording, or camera input to let your companion react to what's happening on your desktop or in your environment.
  </Accordion>
  <Accordion title="Desktop pet mode">
    Switch the companion to transparent background mode to make it appear as a desktop overlay. The window is globally top-most and supports mouse click-through, so you can interact with your desktop normally while your companion is visible.
  </Accordion>
  <Accordion title="Character customization">
    Import your own Live2D models, write custom persona prompts, configure voice cloning, and define expression mappings — all through YAML configuration files without touching any code.
  </Accordion>
</AccordionGroup>

## Get started

<Steps>
  <Step title="Install the prerequisites">
    Install [uv](https://docs.astral.sh/uv/getting-started/installation/) (Python package manager) and [ffmpeg](https://ffmpeg.org/) for audio processing.
  </Step>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/Open-LLM-VTuber/Open-LLM-VTuber.git
    cd Open-LLM-VTuber
    ```
  </Step>
  <Step title="Install dependencies">
    ```bash
    uv sync
    ```
  </Step>
  <Step title="Start the server">
    ```bash
    uv run run_server.py
    ```
    Open your browser to `http://localhost:12393` to start talking with your AI companion.
  </Step>
</Steps>

<Note>
  For detailed platform-specific instructions and GPU setup, see the [Quick Start guide](/get-started/quick-start).
</Note>
