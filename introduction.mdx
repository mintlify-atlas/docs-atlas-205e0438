**Open-LLM-VTuber** is a cross-platform, offline-first AI companion that combines real-time voice conversation, visual perception, and an animated Live2D avatar. All functionality can run locally on your machine — no cloud subscription required.

You can shape it into whatever companion you want: a virtual friend, a debate partner, a character from fiction, or anything else. The persona, voice, and appearance are all configurable.

## What it does

Open-LLM-VTuber connects three types of AI components to create a real-time conversational experience:

- **LLM (Large Language Model)** — processes your messages and generates responses. Supports Ollama, OpenAI, Gemini, Claude, DeepSeek, and many more.
- **ASR (Automatic Speech Recognition)** — converts your voice to text. Supports sherpa-onnx, Faster-Whisper, FunASR, Azure ASR, and others.
- **TTS (Text-to-Speech)** — converts the LLM response back to audio. Supports Edge TTS, MeloTTS, GPT-SoVITS, CosyVoice, Fish Audio, and others.

All three components are swappable through configuration — no code changes required.

## Design philosophy

- **Offline-first**: The default configuration uses locally-running models. Nothing leaves your machine unless you choose a cloud provider.
- **Modular**: Switch LLM, ASR, or TTS backends by changing a single line in `conf.yaml`.
- **Cross-platform**: Runs on Windows, macOS (including Apple Silicon), and Linux. Supports NVIDIA CUDA, Apple MPS, and CPU fallback.
